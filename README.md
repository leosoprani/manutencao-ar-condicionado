# AR RIO | Gerenciador de Manutenção ❄️

Aplicativo premium para gestão de manutenção de ar condicionado, desenvolvido para alta performance e uso offline no Android.

## ✨ Funcionalidades
- **Painel de Bairros**: Visualize as pendências por região do Rio de Janeiro.
- **Gestão de Equipamentos**: Histórico completo e agendamento de próximas visitas.
- **Offline-First**: Funciona totalmente sem internet utilizando IndexedDB.
- **Interface Premium**: Design baseado na estética Obsidian (Glassmorphism + Dark Mode).

## 🚀 Como Iniciar
1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## 📱 Tecnologias
- **Vite**: Build tool ultra rápida.
- **Dexie.js**: Wrapper para IndexedDB (banco de dados local).
- **Material Symbols**: Ícones modernos do Google.
- **CSS3 Moderno**: Variáveis, Flexbox, Grid e Backdrop Filter.
